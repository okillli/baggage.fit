# SEO Guidelines — baggage.fit

## Meta Tags

### useSEO() Hook
Every page/route MUST call `useSEO()` with at minimum:
- `title` — unique per page
- `description` — unique per page (falls back to `siteConfig.description` if omitted)
- `canonical` — full URL for the page

The hook automatically manages cleanup on unmount.

### Required OG Tags
Every `useSEO()` call should include:
- `ogTitle` — can differ from `<title>` for social sharing
- `ogDescription` — concise social description
- `ogUrl` — canonical URL

The hook provides defaults for `ogType` ('website'), `ogImage`, and `twitterCard`.

### Avoid Duplicates
- `index.html` has base OG/meta tags for the SPA shell
- `useSEO()` UPDATES existing meta tags (doesn't create duplicates)
- Do NOT add Twitter Card meta tags in `index.html` — `useSEO()` handles them
- Always check `index.html` before adding new meta tags to avoid duplication

## Structured Data (JSON-LD)

- Home page: `WebSite` schema with `SearchAction` (in `App.tsx`)
- Airline detail pages: `FAQPage` or `Product` schema as appropriate
- Inject via `<script type="application/ld+json" dangerouslySetInnerHTML={...} />`

## Pre-rendering

The Puppeteer pre-render script (`scripts/prerender.mjs`) generates static HTML for SEO crawlers.

### Route Coverage
ALL routes must be in the prerender script:
- Static routes: `/`, `/airlines`, `/about`, `/data-sources`
- Dynamic routes: generated from `airlines.json` via slug function

When adding a new page/route:
1. Add the route to React Router in `main.tsx`
2. Add it to `staticRoutes` in `prerender.mjs`
3. Verify it appears in the generated `sitemap.xml`

### Sitemap Priorities
```
/             → priority 1.0, changefreq weekly
/airlines     → priority 0.8, changefreq weekly
/airlines/*   → priority 0.7, changefreq monthly
other pages   → priority 0.5, changefreq monthly
```

## Canonical URLs

- Every page must have a `<link rel="canonical">` tag
- Use full absolute URLs: `https://baggage.fit/airlines/ryanair`
- The `useSEO()` hook manages this — just pass the `canonical` prop

## Slug Generation

Airline URL slugs are generated by `src/lib/slugs.ts`:
- Lowercase, hyphens for spaces/special chars
- Must be deterministic (same input = same slug)
- Used in both React Router and prerender script
